Using 5 Flex sensors, raspberry Pi and MCP3008(ADC) to realise gesture interface.


Libraries:
bcm2835
QT
Qwt

Make it work:


make distclean
qmake
make
sudo startx ./Gesture

Base on Brend Porr's example of Qwt: https://github.com/berndporr/qwt-example
